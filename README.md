# Proteus

CLI application for Protei IT-School

# Quick Start

```sh
mkdir build && cd build
cmake .. && make
./proteus
```

# Run Tests
```sh
./proteus_tests
```
or
```sh
ctest ./proteus_tests
```

# Crucial Parameters
- -a ADDRESS
- -p PORT
